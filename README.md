# Ultra-lightweight IVIF with only 4.5K parameters
Inspired from [SFDFusion](https://github.com/lqz2/SFDFusion).
## Environments
```
python 3.9.21
cuda 12.7
```

## Train

The training process needs wandb API key.
The config file is `./configs/cfg.yaml`

```
python train.py
```
